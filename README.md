This project focuses on analyzing Adidas sales data to uncover key business insights and trends. The goal of the dashboard is to help understand sales performance, product demand, and regional profitability through interactive visualizations.

The dataset includes information about sales transactions such as product categories, regions, revenue, and profit. Using Power BI, the data was cleaned, transformed, and modeled to create an interactive dashboard that provides meaningful insights for decision-making.

Key Features of the Dashboard:

- Total sales, total profit, and units sold overview
- Sales performance by region and retailer
- Top-performing products and categories
- Monthly and yearly sales trends
- Interactive filters for dynamic data exploration

Tools & Technologies Used:

- Power BI
- Power Query for data cleaning and transformation
- Data modeling and relationships
- DAX for calculated measures
- Data visualization best practices

This project demonstrates skills in data analysis, dashboard design, and transforming raw data into actionable insights for business intelligence.
